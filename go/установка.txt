    # Проверить установленную версию Go
go version

    # Создать новый проект (в папке проекта)
go mod init ваш_проект





    # Основные зависимости Установка
Переменные окружения
go get github.com/joho/godotenv
Для чего: Загрузка конфигурации из .env-файла (API-ключи, пароли).

Веб-фреймворк
go get github.com/gin-gonic/gin 
Для чего: Создание API (роутинг, обработка HTTP-запросов)

Аутентификация
go get github.com/golang-jwt/jwt/v5
Для чего: Работа с JWT-токенами (авторизация пользователей)

Уникальные идентификаторы
go get github.com/google/uuid
Для чего: Генерация UUID для сущностей (пользователи, заказы и т.д.)

websocket
go get github.com/gorilla/websocket
Для чего: Реализация чатов или уведомлений в реальном времени.

Работа с PostgreSQL
go get github.com/jackc/pgx/v5
Для чего: Подключение и запросы к PostgreSQL.

Криптография
go get golang.org/x/crypto
Для чего: Хеширование паролей (bcrypt), шифрование данных.
